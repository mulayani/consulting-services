# Consulting Services Website

A lightweight static website scaffold for a consulting services business.

## Files

- `index.html` - page structure and placeholder copy
- `styles.css` - responsive visual styling
- `script.js` - small dynamic behavior

## Quick Preview

Open `index.html` in a browser.

## Fast Launch Options

The site is static, so it can be hosted on Netlify, Vercel, Cloudflare Pages,
GitHub Pages, or any conventional web host.

## Temporary URL Examples

For the GitHub account `mulayani`, a GitHub Pages URL would look like:

- `https://mulayani.github.io/consulting-services/` for a project repository named `consulting-services`
- `https://mulayani.github.io/` for a special user-site repository named `mulayani.github.io`

For Netlify, a free temporary URL would look like:

- `https://acme-placeholder-business.netlify.app/` if that site name is available
- `https://random-generated-name.netlify.app/` if Netlify assigns a name automatically

Before launch, replace:

- Business name
- Contact email or booking link
- Services copy
- Domain name, if you already own one
